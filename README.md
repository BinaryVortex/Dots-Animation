# Dots Animation

A lightweight, visually pleasing dots/particles animation built with plain HTML, CSS and JavaScript.

![Demo Screenshot](./Screenshot%202024-05-26%20235636.png)

---

## Live demo
Open the `index.html` file in your browser to see the animation locally — no build step or server required. Double-click `index.html` or serve the folder with a simple static server (for example: `npx http-server`).

## Features
- Smooth animated dots/particles on the screen
- Fully client-side: no backend or build tools required
- Small and easy-to-read code (pure JavaScript + CSS)

## Technologies
- JavaScript
- HTML
- CSS

## Getting started
1. Clone the repository:

   git clone https://github.com/BinaryVortex/Dots-Animation.git

2. Open `index.html` in your browser.

That's it — the animation will run immediately.

## File structure
- `index.html` — main page that loads the animation
- `style.css` — styles for the page and canvas
- `script.js` — animation logic and particle system
- `Screenshot 2024-05-26 235636.png` — demo screenshot used above

## Customization tips
- Open `script.js` and look for top-level constants to tweak particle count, size, speed, and colors.
- Use your own background or change CSS variables in `style.css` to adjust the look.
